This is a great tablet for people wanting the Microsoft operating system in tablet form. Basically if you dislike iPad's and find Androids not compatible with your work environment then this is the tablet to get.
Don't expect performance to be up to par with your laptop or desktop. This is made for minimal tasks only. If you think you can photoshop on the subway with this then think again because it just won't happen. Any type of computer gaming isn't going to happen either. It doesn't even support directx.
This is the perfect tablet if your work environment is Microsoft based and you want something to work with at meetings. With the Win 8 Pro upgrade (it comes with standard Win 8), you can join your work domain and like me, get all your network drives and network printers. Ipad users can't do that!
Finding a case for this thing is rough, especially if you have the bigger 60w/hour battery installed. You will have to go with a generic bag.
With the bigger battery (60w/hour), I can go the entire week using this at work without needing to charge. And this is with normal use every single day. At the end of the week it still has plenty of battery life leftover for weekend casual use.
Overall it's good. Wish we could upgrade the ram and it came with a little more powerful processor... maybe an actual non-integrated graphics card. Actually a separate graphics card (with it's own memory) would give this thing the biggest boost to performance.
The USB port supports one device only. Mostly because it only has enough power for one device. You can add a *powered* hub to it though basically making your own docking station. The mini HDMI to full size HDMI cable is easy to find for cheap $$$ if you know where to look.(hide)
Pros:strong glass, extended battery is crazy long
Cons:not too powerful, accessories are hard to find, usb port supports one device only
