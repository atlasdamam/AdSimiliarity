System requirements
To run VIPRE Internet Security 2013 effectively, you need a PC-compatible computer with at least 512 MB of RAM that's running 32- or 64-bit versions of any of the following Windows versions:

  • Windows XP Service Pack 2 or later
  • Windows Vista Service Pack 1 or later
  • Windows 7 or Windows 8

Additionally, we recommend at least 1 GB of memory, a broadband Internet connection and the latest service pack for your version of Microsoft Windows.
