Stimulsoft Reports
Reporting Tools for .NET
Stimulsoft Reports product line is an eminent suite of the .NET platform reporting components. The technologies supported in Stimulsoft Reports are Windows Forms, ASP.NET, MVC, WPF, Silverlight. In the products developers can find the powerful report engine, interactive, intuitive and easy-to-use report designers and viewers. The powerful set of reporting tools allows creating reports of any complexity. Using amazing exporting options users can export reports to Adobe PDF, Microsoft Office etc.
 
Stimulsoft Mobile Reports
Reporting Tools for Mobile Apps
Stimulsoft Mobile Reports products are specifically designed for working on mobile devices. Based on HTML5 and WinRT technologies the products are outstanding set of reporting components for developers who are involved in creating business applications for tablet PCs, smartphones and need to represent data at its best. Powerful suite of controls allows designing reports as complex as needed.


Stimulsoft Reports.Fx
Reporting Tools for Internet
Stimulsoft Reports.Fx products are designed for PHP, Flex and Java technologies. The technologies provide possibility to design, render, view and print reports directly in a web browser. Written in Adobe Flex the report designer and viewer have all the necessary facilities to perform data for a user in any kind. There are no requirements to install additional resources on the server side. Leveraging these reporting components expands the scope of deployment applications for Business Intelligence.
