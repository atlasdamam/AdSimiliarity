eATM tManager Enterprise Appliance Transaction Modules for ControlLogix

eATM tManager Enterprise Appliance Transaction Modules enable the connection of enterprise, business and plant computer systems with popular Allen-Bradley, Siemens and Modicon PLCs/PACs. They enable the exchange of plant floor data via JMS-based messaging services and databases.

The in-chassis modules install in an Allen-Bradley ControlLogix PAC and communicate to other Allen-Bradley controllers via the backplane, bridge modules or EtherNet/IP. They connect to Siemens controllers by Industrial Ethernet® and to Modicon controllers by Modbus® TCP. Any combination of these controllers can be connected to the enterprise via a single eATM module.

Fast and Easy

First, choose the computer adapter (e.g. Microsoft SQL) and drivers for the automation adapter (e.g. Allen-Bradley). Next, install the eATM appliance in a ControlLogix chassis. Then, use the eATM software to configure data exchanges via drag and drop - that's all there is to it. No special languages to know, no special training required, and no additional software packages are required.

Secure

The functionality of eATM modules can be totally independent from existing PAC/PLC programs. In addition, operation can only be changed by authorized personnel using the eATM configuration software. As an industrial product, eATM modules have better security and reduced vulnerability to attack than computer-based options.
