Interested in options trading? View your account alternatives.

Scottrade offers two alternatives for options trading:

Basic Options Trading - If you understand the basics and risks associated with trading options, continue with your application and add options trading to any account type when prompted. This level allows:

  • Buying Puts
  • Buying Calls
  • Selling Covered Calls

Advanced Options Trading - If you are an advanced options trader who requires the latest tools and features go to OptionsFirst (www.options.scottrade.com) to learn more and apply for an advanced options account. This level allows:

  • Cash-secured Puts
  • Spreads
  • Selling Uncovered Options

USA Patriot Act Notice
