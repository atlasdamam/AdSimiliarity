Instructions:


1. Select Firefox at the top left of your Application Menu Bar (for Mac users) browser window.

2. From the drop-down menu, select Preferences (for Mac users) or Options (for PC users).

3. At the top of the function window, select the General tab.

4. Make sure that "Show my home page" is selected for the When Firefox starts dropdown.

5. In the home page text field, type: http://www.theloop.ca
